Figure 2.txt:
Metadata
Sample_ID: Sample ID
Hap_Spp: Pocillopora species or haplotype

Figure 2.vcf:
Linked dataset of 7,887 SNPs

Figure 4 and 6 data.csv:
Metadata
Species.haplotype: Pocillopora species or haplotype
Depth.m: Sampling depth in meters
Site: Sampling site, label corresponds to the site used in the Moorea Coral Reef Long-Term Ecological Research (MCR-LTER) program.
Coral.ID: Coral colony identifier
Type_profile: ITS2 type profile generated by SymPortal
Type_profile_Prop: Proportion of that given ITS2 type profile in colony sampled
Remaining columns: Proportion of ITS2 sequences in colony sampled 

Figure 4 colors.csv:
Metadata
my_colors: Custom colors for each ITS2 sequence
Symbio.clade: ITS2 sequences

Figure 5b - Clad clades.csv:
Metadata
UCI_links: Sample ID that contains Pocillopora species or haplotype, sample ID, and ITS2 type profile
Clad_clades: Clade assignment from figure 5a

Figure 5b_Cladocopium.nex: Nexus tree of Cladocopium taxa in figure 5a

Figure 5b_Pocillopora.nex: Nexus tree of Pocillopora taxa used in PACo analysis

Figure 5b.csv: Matrix of Pocillopora host and Cladocopium symbiont links
